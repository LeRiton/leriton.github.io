Bim, ça vient de vous sauter aux yeux (ou alors, ouvrez les). À droite dans le menu, une petite rubrique Citations avec une citation aléatoire tirée de ce blog. L'occasion pour moi de citer les différents plugins utilisés ici.

C'est parti pour une jolie liste de liens-hommages. Ce blog utilise la plate-forme DotClear. Le thème que j'utilise est 3cols-1fluide version 1.2 de Zoe, que je modifierais peut-être un jour (cruel manque de temps).

Les plugins installés :

  • BBClone - Statistiques, de Znarf, un outil bien pratique pour étudier vos nombreuses visites,
  • Multi Catégories, par TeddyBer, pour associer un billet à plusieurs catégories,
  • GSitemap, de Pep, qui génère une SiteMap pour Google (je le teste actuellement),
  • raisonDeVivre, de lolZim, le plugin dont je vous parlais en intro, qui affiche un message aléatoire.

Merci à tous ces joyeux développeurs.
