Java pas bien

Le 10/10/2005
Catégorie
36 15
Tags
cours programmation

Un diplôme d'info et réseau, c'est bien (pour moi en tout cas). Par contre, un diplôme d'info et réseau où on ne fait que de la prog... C'est quand même nettement plus casse burnes (sans vouloir être vulgaire évidement). En ce moment, vous l'aurez peut-être deviné, on est dans la me... Pardon, dans le Java.

Le sujet du projet est simple, voir trivial (un mot qu'il est bien pour les profs, voir un précédent billet). Je ne résiste pas à l'envie de en vous mettre (en tout bien tout honneur) les premières lignes :

FourmIR2000 est un jeu de simulation de fourmilière. Deux fourmilières (une rouge et une noire) de la forêt Veur vivent trop proche et bêtement se font la guerre depuis maintenant quelques générations afin d'être les seules à régner sur le territoire.

Le truc, c'est que le super jeu dont ils nous parlent... C'est nous qu'on doit le faire. La bonne nouvelle ! Vous comprendrez donc les périodes de disettes de billets sur ce blog, les soirées sont courtes (et les nuits sont pleines de cauchemars à base de fourmis qui me montent dessus, me rentrent dedans par tous les orifices et me dévorent de l'intérieur... Le tout en énumérant les meilleurs passages de mon bouquin de Java).

Une petite classe, pour ceux qui veulent comprendre l'état avancé de désespoir qui m'habite (vous avez vraiment des idées mal placées) : 


import java.util.Random;
public class RitonMark {
    public static void main (String args[]) {
        System.out.println("LeRiton obtient "+getRitonMark()+"/20 a son fuckin projet.");
    }
    public int getRitonMark() {
        Random rand = new Random();
        return rand.nextInt(20);
    }
}

Sur ce, j'y retourne. Un conseil pour les jeunes et insouciants étudiants qui passeraient par là : sauf si vous êtes un peu timbrés, évitez la prog, ça rend cinglé et c'est très dangereux pour les réserves de café mondiales.
